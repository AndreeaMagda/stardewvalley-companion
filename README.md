# 🌾 Stardew Companion

A cozy web app to help manage your Stardew Valley farm — track crops, resources, villager birthdays, and seasonal events.

**Stack:** React 19 · TypeScript · Vite · Tailwind CSS v4 · Zustand · Supabase

## Setup

1. Run [`schema.sql`](./schema.sql) in your Supabase SQL editor
2. Copy `client/.env.example` → `client/.env` and add your Supabase URL and anon key
3. `npm install && cd client && npm run dev`
